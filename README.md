# checkIN - Face-Recognition_Engage-2022
This is an Android Application developed for Microsoft Intern Engage 2022 Programme.<br />
It's an Smart Attendance App that is powered by face recognition technology.<br />
The UI/UX is really simple and self explanatory, the user will not face any issues.<br />

## 📕 Resources
- Video demo link - [YouTube](https://www.youtube.com/watch?v=toJiBPxem5I&ab_channel=LitCoder)

## 💻 Tech Stack used 
- **Flutter** - Developed using Android studio and Java as developing Language
- **Figma** - [**Designed using Figma**]
- **Firebase** - To authenticate users, store user's data and perform operations.

## ✨ Features
- **Login** - Allows people to log in using credentials provided by their organisation.
- **View their checkIn/checkOut status** - Shows the current date and time along with time of checkIn
- **Mark Attendance** - Users just have to slide a bar to mark their attendance
- **Capture Images** - Allows users to capture Images
